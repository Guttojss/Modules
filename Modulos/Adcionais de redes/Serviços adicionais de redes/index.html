Breve Introduçao

Os serviços adicionais de rede são diversos recursos que facilitam o compartilhamento e gerenciamento de informações entre dispositivos numa rede.

    Alguns exemplos :
  • Firewall
  • Proxy
  • DNS

    • File Servers

    É um tipo de servidor cuja função é guardar e gerenciar dados, para qualquer dispositivo consiga aceder e partilhar os dados. Uma vez que este tipo de servidor contem dados este tende a ser vitima de muitos ataques, tais como ransomware, logo precisam ter algum nível de segurança.

      Alguns dos objetivos sao :
  • Facilitar a organização e acesso aos dados
  • Automatizaçao de Backups

    • Print Servers

    É um tipo de servidor cuja função é gerenciar impressões, ou seja gerência a prioridade das impressões, mostra informações sobre as impressões. este tipo de servidor também pode ser usado em casa para melhorar a experiencia de impressão.

      Alguns objetivos sao :
  • Organizar o tráfico de impressoes

    • Mail Servers

    É um tipo de servidor cuja função é gerenciar e-mails, guardar e-mails , receber e enviar e-mails. este pode ser divido em 2 secções, mail servers que recebem e que enviam. os mail servers funcionam através de protocolos, os mais frequentes sendo : SMTP, para enviar email e IMAP, para receber o email.
