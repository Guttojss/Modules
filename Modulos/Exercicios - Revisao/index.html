C e C++

  • Quais são as principais diferenças entre as linguagens de programação C e C++?

    • C++ é uma extensão de C que inclui recursos adicionais, como programação orientada a objetos. C++ mantém a compatibilidade com C, mas oferece algumas funcionalidades a mais

  • Dê exemplos de situações em que cada uma dessas linguagens seria mais apropriada

    • C é mais usado para sistemas operacionais e desenvolvimento de baixo nível. C++ é mais adequado para projetos que precisam de programação orientada a objetos, como por exemplo desenvolvimento de jogos.

Alocação de Memória

  • Como a alocação de memória é tratada em C e C++?

    • Em C, você utiliza malloc() e free() para alocação e liberação de memória dinâmica enquanto que em C++, pode-se usar new e delete

  • Existe alguma diferença significativa entre os dois?

    • A principal diferença é que em C++, o uso de new e delete é preferível, pois eles chamam automaticamente os construtores e destrutores dos objetos.

Comandos SQL para CRUD

  • Liste os comandos SQL básicos para realizar operações CRUD (Create, Read, Update, Delete) em um banco de dados

    • Create:
    INSERT INTO Carros (matrícula,marca)
    VALUES (78-XD-15,Tesla);

    Read:
    SELECT * FROM carros
    WHERE ano=2001;

    Update:
    UPDATE carros
    SET cor= preto
    WHERE matricula=”70–DX-95”;
    Delete:
    DELETE FROM carros
    WHERE vendido=true;
