Oque são os utilitários?

Utilitários são, software para gestão e manutenção de um sistema operativo, como por exemplo um software para limpar disco rigidos, este tipo de programas tende a ser um pouco mais técnicos, por isso vou fazer uma breve explicação :

Gestores de Disco

Programas cuja função seja, limpar espaço no disco rígido tendem a ser chamados de Gestores de discos, estes programas são usados para apagar ficheiros inuteis no computador assim melhorando o desempenho e libertando espaço no disco. alguns softwares que possuem esta função são :

  • CCleaner
  • DiskGenius
  • Wise Disk Cleaner

    • Antivírus

    Programas cuja função seja, proteger o teu computador de possiveis malwares, são denominados por AntiV´rius, estes programas tendem a analisar todos os ficheiros do teu computador em busca de possiveis ficheiros com algum malware, isto é, em busca de algo que possa danificar o teu computador ou os teus dados.os antivírus mais comuns são :

  • Avast
  • McAfee
  • Eset

    • Conversores de ficheiros

    Programas cuja função seja,converter um tipo de ficheiro para outro tipo, são normalmente conhecidos como conversores de ficheiros, este tipo de software é mais comum online, alguns exemplos do uso deste tipo de programas é, para converter .jpg para .png, entre outros. alguns dos sites mais conhecidos para este tipo de operação são :

  • PNG to JPG
  • cloudconvert
  • convertio
