Este Fluxograma de uma base de dados de uma biblioteca, possuimos 3 tabelas, Livro,Cliente e Compra. Cada tabela possui vários atributos,estes são, algumas informações que as tabelas possuem.

No caso do Livro, estes dados são :

  • Id_Cliente
